# Аналитическая лабораторная работа №1 AWS. Варинат 11
![image](https://github.com/NorthPole0499/Cloud_Technologies/assets/76227471/77b4232a-c422-475c-9c43-ed182ca054aa)

### Цель работы:

Знакомство с облачными сервисами. Понимание уровней абстракции над инфраструктурой в облаке. Формирование понимания типов потребления сервисов в сервисной-модели. 
Сопоставление сервисов между разными провайдерами. Оценка возможностей миграции на отечественные сервисы.

### Дано:

1. Слепок данных биллинга от провайдера после небольшой обработки в виде SQL-параметров. Символ % в начале/конце означает,
   что перед/после него может стоять любой набор символов.
2. Google с документациями провайдера

### Ход работы

Необходимо импортировать файл .csv в Excel и определить соответствие каждого сервиса международного провайдера русскому сервису. 
Заполнить информацию об отечественных сервисах, сопоставляя их напротив AWS сервисам. 

Как было сказано выше, работа происходила в Excel, где оперировать данными из csv файла было достаточно удобно. К тому же, команда решила отказаться от использования модели Apptio TBM Unified Model (ATUM) и не заполнять первые четыре пустых столбца в силу четкой и ясной документации как у Amazon, так и у крупных российских облачных компаний.

Так что сначала брался продукт компании Amazon, а далее, вручную находился его аналог на российском рынке.

### Описание сервисов Amazon
В следующей таблице опишем встретившиеся во время выполенения работы сервисы компании Amazon.

| Сервис | Описание|
| --- | --- |
| Amazon Athena | Бессерверный интерактивный аналитический сервис, построенный на фреймворках с открытым исходным кодом, который позволяет анализировать петабайты данных там, где они находятся. Сервис основан на принципах работы с языком запросов SQL, а настраивать инфраструктуру или управлять ею не требуется. |
| Amazon S3 | Объектное хранилище, рассчитанное на хранение и извлечение любых объемов данных из любой точки сети. S3 – это простой сервис хранилища, который отличается самой высокой надежностью и практически неограниченной масштабируемостью при очень низких затратах. Возможности сервиса включают в себя хранение и извлечение объектов, контроль доступа, репликацию данных и многое другое. |
| APN Fee | Классификация, одобренная AWS для глобального сообщества поставщиков облачных услуг и вендоров после того, как они пройдут определенные тесты. Одобренные партнеры AWS могут участвовать в предоставлении облачных сервисов и решений AWS для клиентов, в то время как AWS предлагает технологическую платформу и маркетинг, необходимые для создания, внедрения и сбыта этих решений. Для всех уровней партнерской сети AWS требуется ежегодная плата за APN в размере 2500 долларов. |
| Amazon App Stream | Cервис вычисления для конечных пользователей AWS, который можно настроить для потоковой передачи приложений SaaS или доставки виртуальных рабочих столов с выборочным сохранением данных. Он позволяет пользователям получать доступ к своим приложениям и рабочим столам из любого места, где есть доступ в интернет. AppStream также предоставляет возможность масштабирования ресурсов в зависимости от потребностей пользователя. |
| Amazon Lex | Сервис, который позволяет создавать голосовых помощников для приложений, ботов для мессенджеров и других устройств. Он использует технологии искусственного интеллекта и машинного обучения для обработки естественного языка, чтобы понимать и отвечать на запросы пользователей. |
| Amazon Lightsail | Сервис для создания и управления виртуальными серверами. Он предлагает простую и доступную альтернативу традиционным выделенным серверам, позволяя пользователям быстро и легко создавать свои собственные серверы в облаке. Lightsail также предоставляет инструменты для управления ресурсами сервера, такие как масштабирование ресурсов, мониторинг производительности и управление доступом. |
| Amazon Neptune | Бессерверная графовая база данных с превосходной масштабируемостью и доступностью. В базу данных Neptune встроены функции безопасности, непрерывного резервного копирования и интеграции с другими сервисами AWS. Также, имеется аналитическое ядро базы данных, позволяющее быстро анализировать большие объемы графовых данных для получения аналитики и выявления тенденций. |
| Amazon Managed Blockchain | Полностью управляемый сервис (блокчейн-платформа), призванный помочь создавать отказоустойчивые приложения Web3 как в публичных, так и в частных блокчейнах. Содержит инструменты и подсервисы для создания, тестирования и развертывания блокчейн-приложений. |
| Amazon Maice | Cервис, который обеспечивает безопасность и конфиденциальность, используя машинное обучение (ML) и сопоставление со специальными шаблонами для обнаружения и защиты конфиденциальных данных. |
| Amzon ML | Cервис, который предоставляет инструменты и технологии для создания, тренировки и развертывания моделей машинного обучения. Он включает в себя инструменты для подготовки данных, выбора модели, оценки качества модели и развертывания модели в production. С помощью Amazon ML, разработчики и data scientists могут создавать высокоточные модели машинного обучения без необходимости написания кода или настройки инфраструктуры. |
| AWS Config | Инструмент конфигурации, который позволяет организациям управлять, отслеживать и контролировать изменения в своих AWS-ресурсах. AWS Config обеспечивает централизованное управление конфигурациями, автоматическое обнаружение изменений и управление правилами. |
| AWS Cost Explorer | Обозреватель затрат AWS предлагает удобный интерфейс для визуализации, анализа затрат и использования ресурсов AWS, а также для управления этими параметрами в долгосрочной перспективе. Имеется возможность быстро приступать к работе с помощью пользовательских отчетов для анализа расходов и использования. |
| AWS Global Accelerator | Cервис ускоренной доставки контента, который помогает ускорить загрузку веб-сайтов и приложений для клиентов по всему миру. Он использует сеть распределенных точек присутствия (POP) AWS и технологию локального кэширования для оптимизации доставки контента клиентам, позволяя разработчикам и владельцам сайтов определять географические регионы, для которых требуется ускорение, и автоматически оптимизировать доставку контента на основе этих настроек. |
| AWS Glue | Бессерверный сервис интеграции данных, который упрощает поиск, подготовку, перемещение и интеграцию данных из множества источников для анализа, машинного обучения и разработки приложений. А именно, сервис использует Apache Spark для обработки данных и позволяет создавать ETL-процессы (извлечение, преобразование и загрузка данных). |
| AWS Greengrass | Периферийная среда выполнения и облачный сервис с открытым исходным кодом для создания, развертывания и управления ПО устройств. Платформа позволяет разработчикам создавать масштабируемые и надежные IoT-решения. Предоставляет собой набор сервисов и инструментов для управления устройствами, обработки данных, безопасности и тд. |

### Выводы о возможностях миграции

Amazon несомненно является мировым гигантом облачных сервисов. Не всем представленным сервисам в нашей таблице нашлись аналоги на российском рынке. Однако, хочется сказать, что у основных игроков отечественного рынка облачных технологий, то есть у Yandex, Selectel и Sber, есть преимущества, имея которые возможно тягаться за первенство с Amazon. 

К примеру, что у Amazon, что у Yandex идеально прописанная документация. Также, у компании Yandex есть аналоги почти всех основных сервисов Amazon. Единственный минус, который можно отметить, так это - недостаток времени для развития. Большое количество российских сервисов пока что слишком недоделанные или вообще находятся в стадии тестирования. Но можно с уверенностью сказать, что в ближайшие пару лет будет возможно мигрировать с сервисов Amazon, например, на сервисы того же Yandex.
